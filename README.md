# Flutter App (starter)
This is a minimal Flutter app for login/register that talks to the Laravel API.

How to run:
1. Install Flutter SDK.
2. Open the `flutter_app` folder.
3. Run `flutter pub get`.
4. Start an Android emulator (or use a device). For Android emulator, use API URL http://10.0.2.2:8000
5. Run `flutter run`.
